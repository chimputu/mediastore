# 📁 MediaStore

A modern, full-featured media management platform built with Next.js 16, Clerk Authentication, Supabase, and Cloudinary.

![MediaStore Banner](https://via.placeholder.com/1200x400/000000/FFFFFF?text=MediaStore)

## ✨ Features

- 📸 **Upload & Manage Media** - Upload images, videos, and audio files with ease
- 🎬 **Video Preview** - Hover over videos to preview playback with sound controls
- 🔍 **Search & Filter** - Quickly find your media with search and category filtering
- 📱 **Responsive Design** - Works seamlessly on desktop, tablet, and mobile
- 🔐 **Secure Authentication** - Powered by Clerk for secure user management
- ☁️ **Cloud Storage** - Files stored securely on Cloudinary
- 📊 **Storage Management** - Visual storage usage bar with limits
- 🎨 **Clean UI** - Modern, intuitive interface with dark/light support

## 🚀 Tech Stack

| Technology | Purpose |
|------------|---------|
| [Next.js 16](https://nextjs.org/) | React framework with Turbopack |
| [Clerk](https://clerk.com/) | Authentication & User Management |
| [Supabase](https://supabase.com/) | Database & Storage |
| [Cloudinary](https://cloudinary.com/) | Media Storage & Optimization |
| [Tailwind CSS](https://tailwindcss.com/) | Styling |
| [TypeScript](https://www.typescriptlang.org/) | Type Safety |
| [Lucide Icons](https://lucide.dev/) | Icon library |
